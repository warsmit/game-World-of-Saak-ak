2D игра с использование библиотеки SFML
колличество игроков 1-4
перед запуском необходимо подключить 1-4 джостика
для победы необходимо 5 убийств противника
суицид даёт -1
---------------------------------------
Сборка осуществленна в Qt Creator
Компилятор MinGW (с линковкой sfml-graphics-d;sfml-window-d;sfml-system-d;sfml-audio-d;)
На платформе win7x64
