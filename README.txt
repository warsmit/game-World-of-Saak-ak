---------------------------------------
2D игра с использование библиотеки SFML
колличество игроков 1-4
перед запуском необходимо подключить 1-4 джостика
для победы необходимо 5 убийств противника
суицид даёт -1
---------------------------------------
Запуск через Qt Creator
-распаковать в диск C архив SFML.zip (C:\SFML)
-запуск ClientGameMax.pro файла
---------------------------------------
в папке debug есть .exe файл, тестировалось на win7x64
Build system: qmake
Kits - Desktop Qt 5.5.1 MinGW 32bit
---------------------------------------
